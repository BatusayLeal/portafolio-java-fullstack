# portafolio-java-fullstack
Portafolio Java Fullstack con Spring Boot 4.1. 5 proyectos profesionales que demuestran autenticación segura (Google, Azure Entra ID, AWS, AD/LDAP), conexiones a bases de datos locales y cloud con SSL/TLS, webservices REST y consumo de APIs externas. Código limpio, arquitectura clara y perfil híbrido de Infraestructura + Desarrollo.
