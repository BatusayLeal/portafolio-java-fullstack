## 📋 Estructura de cada proyecto

Todos los proyectos siguen la misma estructura profesional:

- `usr_connect/` → Conectores de autenticación (Google, Azure, AWS, AD/LDAP)
- `bds_connect/` → Gestión de conexiones a bases de datos (local sin SSL ↔ cloud con SSL/TLS)
- Código limpio, capas bien definidas y documentación completa
